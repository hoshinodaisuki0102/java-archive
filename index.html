Java란?

Java란 무엇인가요?

Java는 1995년에 등장한 객체지향 프로그래밍 언어로, 지금도 다양한 분야에서 널리 사용되는 안정적이고 강력한 언어입니다.

  • 역사 — 1995년 Sun Microsystems에서 등장
  • 기능 — 객체지향, 자동 메모리 관리, JVM 기반 실행
  • 쓰임새 — 서버, 금융, 안드로이드, 게임 서버 등
  • 장점 — 안정성, 확장성, 문서 풍부, 대규모 프로젝트에 강함
