01. 첫 번째 자바 프로그램

자바(Java)는 모든 프로그램이 반드시 main 메서드에서 시작됩니다. 즉, 컴퓨터가 프로그램을 실행할 때 가장 먼저 들어오는 문이 바로 main()이라고 생각하면 됩니다.

자바 프로그램의 기본 구조

가장 기본적인 형태는 아래와 같습니다:

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}

지금은 이 구조를 모두 이해하려고 애쓰지 않아도 됩니다. “코드는 이 네모 안에 들어간다” 정도만 기억해도 충분합니다.

  • public class Main { ... } → 프로그램의 기본 틀, 'Main'이라는 이름의 클래스
  • public static void main(String[] args) → 프로그램의 시작 지점
  • System.out.println() → 콘솔(출력창)에 글자를 표시하는 코드

오늘의 목표

자바의 첫 번째 목표는 “내가 원하는 문장을 화면에 출력하는 것”입니다. 이것만 할 수 있어도 자바의 절반은 이해한 셈입니다!
