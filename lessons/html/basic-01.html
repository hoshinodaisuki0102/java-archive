기본기 1강 — 프로그래밍이란?

프로그래밍은 컴퓨터에게 해야 할 일을 순서대로 알려주는 과정입니다. 사람은 상황을 보며 유추하거나 대충 판단할 수 있지만, 컴퓨터는 스스로 판단하지 못합니다. 따라서 컴퓨터는 정확한 명령을, 정확한 순서대로 받아야만 움직입니다.

예를 들어 “라면 끓여”라는 말만 들으면 사람은 자연스럽게 순서를 떠올립니다. 하지만 컴퓨터에게는 아래와 같이 한 줄 한 줄 자세하게 알려주어야 합니다.

1. 냄비를 준비한다. 2. 냄비에 물 500ml를 넣는다. 3. 가스레인지 불을 켠다. 4. 물이 끓으면 스프를 넣는다. 5. 면을 넣고 3분간 끓인다.

이처럼 프로그래밍은 아주 자세하고 명확한 순서를 “문법”에 맞추어 적는 것입니다.

프로그래밍 언어란?

사람은 한국어·영어로 말하지만, 컴퓨터는 이런 언어를 이해하지 못합니다. 컴퓨터가 이해할 수 있도록 구조화된 언어가 바로 프로그래밍 언어입니다.

“프로그래밍 언어는 컴퓨터가 이해할 수 있는 규칙을 가진 언어이다.”

왜 자바(Java)를 배우는가?

자바는 전 세계적으로 널리 사용되는 언어이며, 문법이 비교적 일정하고 명확해서 기초를 탄탄하게 쌓기 좋습니다.

  • 문법이 명확해 초보자가 기초를 정확히 익히기 좋다.
  • 기업·공공기관·은행 등에서 널리 사용된다.
  • 마인크래프트(Java Edition)도 자바로 만들어졌다.

자바의 첫 명령문 — println

자바에서 화면에 글자를 출력하는 가장 기본적인 명령은 println입니다. 아직 모든 문법을 이해하지 못해도 괜찮습니다. “이렇게 쓰면 글자가 한 줄 나온다” 정도만 먼저 익히면 됩니다.

System.out.println("Hello Java");
실행 결과
Hello Java

읽는 순서를 차근차근 보면 다음과 같습니다.

  • System : 자바가 제공하는 여러 기능 묶음.
  • out : 그 중에서 “출력(바깥으로 보여주기)” 기능.
  • println : 한 줄을 출력한 뒤 줄을 바꾼다.
  • ("Hello Java") : 출력할 실제 내용.
  • ; : “여기까지가 한 문장이다”라는 표시.

실습 2 — 문장을 바꾸어 출력하기

이번에는 출력 내용을 바꾸어 봅시다. 콘솔에 다음 문장이 한 줄 출력되도록 코드를 작성해 보세요. 
프로그래밍 첫 걸음!
출력 결과가 여기에 표시됩니다.

실습 3 — 두 줄을 차례대로 출력하기

아래와 같이 두 줄이 순서대로 출력되도록 코드를 작성해 보세요. 
Hello Java
나는 자바를 배우는 중입니다.
아직 실행하지 않았습니다.
오늘은 프로그래밍이란 무엇인지, 그리고 자바에서 가장 기본이 되는 출력 명령 System.out.println의 역할을 살펴보았습니다. 컴퓨터는 애매한 지시를 이해하지 못하므로, 우리는 정확한 문법과 순서를 지켜 한 줄 한 줄 명령을 써야 합니다. 이제 “이 코드를 쓰면 화면에 글자가 한 줄 나온다”라는 감각을 충분히 익혀 두는 것이 중요합니다.
