기본기 2강 — println 완전 정복

1강에서 System.out.println("Hello Java"); 한 줄을 사용해 보았습니다. 이제 이 한 줄을 이루는 요소를 조금 더 천천히 뜯어서 살펴보겠습니다.

세미콜론(;)은 왜 붙을까?

자바에서는 한 줄짜리 명령의 끝에 세미콜론(;)을 붙여야 합니다. 세미콜론은 “여기까지가 한 문장이다”라는 표시입니다. 한국어에서 마침표 역할과 비슷합니다.

System.out.println("Hello"); System.out.println("Java");
실행 결과
Hello Java

만약 세미콜론을 빼먹으면 자바는 “문장이 어디에서 끝나는지” 알지 못해 문법 오류(compile error)를 내게 됩니다.

괄호와 큰따옴표

println 뒤의 괄호와 큰따옴표도 중요합니다.

  • 괄호 ( ) 안에는 “출력할 대상을 넣는다”.
  • 문자(글자)를 출력하고 싶을 때는 큰따옴표 " "로 감싼다.
System.out.println("123"); System.out.println(123);
실행 결과
123 123

겉보기에는 둘 다 같은 결과가 출력되지만, 첫 번째 줄은 “문자열”, 두 번째 줄은 “숫자”라는 점에서 차이가 있습니다. 지금은 글자를 출력할 때는 꼭 큰따옴표로 감싼다 정도만 기억해 둡시다.

println과 print의 차이

자바에는 println과 비슷한 친구인 print도 있습니다.

System.out.print("Hello"); System.out.print("Java");
실행 결과
HelloJava

print는 출력 후에 줄을 바꾸지 않기 때문에, 두 문장이 한 줄에 이어서 붙어 나옵니다. println은 출력 후 자동으로 줄을 바꾼다는 점만 기억해 두면 충분합니다.

이번 강의에서는 System.out.println 한 줄에 들어 있는 세미콜론, 괄호, 큰따옴표의 의미를 자세히 살펴보고, print와 println의 차이도 함께 연습했습니다. 출력 명령의 기본 구조를 정확히 알고 있으면 이후에 변수, 조건문, 반복문을 배울 때도 “내가 지금 무엇을 화면에 보여 주고 있는지”를 명확하게 파악할 수 있습니다.
